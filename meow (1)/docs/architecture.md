# Архитектура

## Контейнеры

- `app` — FastAPI, frontend, агент, валидация и песочница Lua
- `ollama` — локальная модель
- `ollama-init` — автозагрузка модели при первом старте

## Поток запроса

1. UI отправляет `prompt` и `context` в `POST /api/generate`
2. `LuaAgentService` делает проверку полноты запроса
3. Генератор получает финальный prompt
4. `LuaTools.check_syntax()` запускает `luac -p`
5. `LuaTools.run_in_sandbox()` выполняет код в изолированной среде
6. Результат, ошибки и шаги итераций сохраняются в SQLite и возвращаются в UI

## Безопасность

- блэклист опасных Lua API
- отдельный subprocess для Lua
- timeout
- лимит памяти
- только ограниченный набор глобалов
